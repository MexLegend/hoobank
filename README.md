## Cloning the repository

```shell
git clone https://github.com/MexLegend/hoobank.git
```

## Getting Started

Install the dependencies:

```sh
$ pnpm i
# or
$ npm i
```

Start the app

```sh
$ pnpm dev
# or
$ npm run dev
```

## Built With

- React 18
